##  Descripción

**PetLove** es una aplicación desarrollada en **.NET** con **C#** y base de datos **PostgreSQL**, enfocada en la conexión entre dueños de mascotas interesados en la cruza responsable de animales. La plataforma permite gestionar perfiles de mascotas, encontrar coincidencias compatibles y facilitar el contacto entre usuarios.

El objetivo del sistema es promover una cruza controlada y organizada, facilitando la interacción entre propietarios de mascotas con intereses similares.

---

## ❗ Problemática

En muchos casos, la cruza de mascotas se realiza de manera informal, sin información adecuada sobre compatibilidad, historial de salud o características de los animales. Esto puede generar problemas de salud, falta de control y dificultad para encontrar coincidencias adecuadas.

Además, no existe una plataforma centralizada que permita a los dueños de mascotas conectar de forma segura y organizada para este propósito.

---

##  Solución Propuesta

Se desarrolló una aplicación que permite:

- Registro de usuarios  
- Creación de perfiles de mascotas  
- Publicación de mascotas disponibles para cruza  
- Búsqueda de coincidencias según características  
- Comunicación entre usuarios  
- Gestión de información de cada mascota  

La plataforma centraliza la información y facilita la conexión entre usuarios interesados en la cruza responsable de mascotas.

---

##  Arquitectura

El sistema está basado en una **arquitectura cliente-servidor**.

- **Frontend / Backend:** Aplicación desarrollada en **.NET**
- **Lógica de negocio:** C#
- **Base de Datos:** PostgreSQL

La aplicación sigue un enfoque estructurado utilizando capas para separar la lógica de negocio, acceso a datos y presentación.

---

##  Desarrollo del Proyecto

El desarrollo inició con el análisis de los requerimientos del sistema y la definición de entidades principales como usuarios y mascotas.

Posteriormente se diseñó la base de datos en PostgreSQL, estableciendo relaciones entre usuarios, perfiles de mascotas y publicaciones.

En la implementación se utilizó .NET con C# para construir la lógica del sistema, así como la comunicación con la base de datos mediante consultas estructuradas.

Finalmente se realizaron pruebas funcionales para validar el registro de usuarios, la gestión de mascotas y la interacción entre perfiles.

##  Resultados Obtenidos

Se logró desarrollar una aplicación funcional capaz de gestionar usuarios y mascotas, permitiendo la interacción entre ellos para fines de cruza responsable.

---

##  Impacto

PetLove contribuye a la digitalización del proceso de cruza de mascotas, brindando una plataforma estructurada que ayuda a evitar prácticas informales y promueve una mejor organización entre dueños de animales.

Además, el proyecto fortaleció conocimientos en desarrollo con .NET, diseño de bases de datos relacionales en PostgreSQL y construcción de aplicaciones orientadas a la gestión de información.
